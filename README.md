# Bright-Learn-SQL-Exercises
This repository provides answers and hand written queries to Bright Learn SQL exercises covering SQL concepts and fundamentals, aggregate functions, case statements and joins. The repository contains a collection of structured SQL exercises designed to build foundational to intermediate SQL skills. The exercises are based on practical datasets and focus on writing queries manually (handwritten logic) and visualising expected output tables.

The repository is organised into four core exercises:
 SQL Fundamentals,
 Aggregate Functions & Grouping,
 CASE Statements,
 Joins, UNION, Filtering, and Aggregates
